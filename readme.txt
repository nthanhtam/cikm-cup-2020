The Starting Kit contains:
	-simple_solution_validation.zip --- an example of a ready to use zip file for the validation phase
	-simple_solution_test.zip --- an example of a ready to use zip file for the test phase

Each submission has be a "zip" file. The name of the "zip" file can be used to distinguish different submission, e.g., xgboost_1_simple_solution.zip.
Depending on the type of the submission (validation or test) the "zip" file has to contain a text file with the name validation.predict or test.predict.
